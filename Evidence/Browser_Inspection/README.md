# Browser Inspection Evidence

## Overview

This folder contains screenshots collected using the Firefox Browser Developer Tools while inspecting the JSONPlaceholder API.

The inspection focused on analyzing HTTP requests, response headers, caching behavior, server information, and API responses without modifying or attacking the service.

## Contents

- JSON response inspection
- Single user endpoint response
- Photos endpoint response
- HTTP response headers
- Cache-Control header
- Connection header
- Server header
- Browser Network analysis

## Purpose

These screenshots provide evidence used during the API Security Risk Analysis and support the findings documented in the final report.

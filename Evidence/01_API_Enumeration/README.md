# API Enumeration

## Description

This folder contains screenshots of the API enumeration process performed on the **JSONPlaceholder** public API.

The evidence demonstrates the identification and inspection of available API endpoints and their responses using a web browser. These screenshots were used to analyze the structure of the API, response data, and publicly accessible resources as part of the API Security Risk Analysis.

## Files

- **01_Posts_API_Response_Start.png** – Beginning of the `/posts` endpoint response.
- **02_Posts_API_Response_End.png** – End of the `/posts` endpoint response.
- **03_Users_API_Response.png** – Response from the `/users` endpoint.
- **04_Single_User_API_Response.png** – Response from the `/users/1` endpoint.

## Objective

- Enumerate available API endpoints.
- Review publicly accessible data.
- Identify potential data exposure.
- Support the overall API security assessment.

> **Note:** All testing was performed on the public JSONPlaceholder test API using read-only requests for educational purposes.
